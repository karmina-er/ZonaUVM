# ZonaUVM

## Descripción 
ZonaUVM es una página web como guía de supervivencia estudiantil que reúne en un solo lugar herramientas como un chatbot con consejos y un mapa del campus, además de enlaces directos a plataformas como Conexión Lince, Blackboard y Pórtico, para facilitar la organización y la vida cotidiana del estudiante.

## Funcionalidades Principales
### index.html
Navegación Centralizada: Actúa como un hub que conecta todos los módulos mediante tarjetas de servicio interactivas.

Interfaz Adaptativa: Implementa un diseño visual moderno con efectos de desvanecimiento (fade-in) y una estética coherente con la identidad institucional.

### mapa.html
Croquis Dinámico: Renderiza un mapa donde cada aula o área es interactiva y reacciona al cursor.

Gestión de Incidencias: Permite a los usuarios reportar problemas técnicos o de infraestructura directamente sobre el mapa.

Cálculo de Rutas: Incluye lógica para trazar caminos entre puntos específicos del campus.

### menu.html
Filtrado de Categorías: Permite a los estudiantes filtrar el menú por tipo de alimento.

Modo Administrador: Incluye una interfaz protegida para la edición de productos y gestión del inventario.

### horario 5.html
Gestión de Clases: Permite añadir, visualizar y eliminar materias dentro de una cuadrícula semanal dinámica.

Persistencia Local: Guarda la información del horario en el navegador para que no se pierda al cerrar la sesión.

Indicadores de Tiempo: Resalta visualmente la clase actual y el tiempo restante de la sesión.

### chatbot_de_consejos.html
Interacción en Tiempo Real: Procesa mensajes del usuario y genera respuestas automáticas basadas en consejos académicos y de vida universitaria.

Interfaz de Chat: Sistema de mensajes con burbujas diferenciadas para usuario y bot, incluyendo indicadores de escritura.

### directorio.html
Tarjetas de Contacto: Genera dinámicamente fichas con información de coordinadores, incluyendo correos y extensiones.

Búsqueda Rápida: Implementa filtros para encontrar personal específico por departamento o nombre.
